# vagrant-nvm-rvm-psql
Vagrant box with:
* nvm for Node.js
* rvm for ruby
* PostgreSQL 9.4
* git
* etc.

## require
Vagrant
https://www.vagrantup.com/downloads.html


## install
clone repo
cd folder

vagrant up

## notes
